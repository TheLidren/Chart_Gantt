# Диаграмма Ганта
Диаграмма Ганта — это инструмент просмотра график работ по проекту, который строится в виде таблицы с этапами и сроками выполнения.

### Содержание
- [Назначение](#why)
- [Установка](#install)
- [Применение](#apply)
- [Лицензия](#license)

#### Назначение:

<a name="why"></a>
Данный проект автоматически заполняет Excel-файл на основе результата sql-запроса MS SQL.
Диаграмма показывает:
- продолжительность выполнения каждой задачи;
- время, которое занимает каждая задача;
- ситуации, когда время выполнения одной задачи превышает установленные сроки.
В результате, диаграмма Ганта позволяет оценить все выполненные задачи и определить, какие конкретно задачи необходимо оптимизировать.

#### Применение:

<a name="apply"></a>
Внедрение проекта позволило:
